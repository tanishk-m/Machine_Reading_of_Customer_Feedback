# Machine Reading - Sentiment Analysis Platform

**A powerful AI-driven platform** for analyzing customer feedback from both **text and audio reviews**. It extracts sentiment (Positive, Negative, Neutral), generates actionable business insights using AI, visualizes trends, and automatically sends detailed email summaries.

Built to help businesses turn raw customer reviews into meaningful, actionable intelligence.

👤 **Author**: Tanishk Meena

---

## ✨ Key Features

- **Multimodal Sentiment Analysis**: Analyze both **text** and **audio** reviews (Positive / Negative / Neutral)
- **AI-Powered Insights**: Generate smart, actionable business recommendations using Google Gemini
- **Interactive Visualizations**: Sentiment trends over time + keyword clouds
- **Automated Email Notifications**: Receive comprehensive analysis summaries instantly
- **Secure Authentication**: Full signup/login system with Supabase
- **Modern & Responsive UI**: Clean, intuitive interface

---

## 🛠️ Tech Stack

| Layer       | Technologies                              |
|-------------|-------------------------------------------|
| **Frontend** | React, Vite, Tailwind CSS                |
| **Backend**  | Node.js, Express                         |
| **Database** | Supabase (PostgreSQL + Auth)             |
| **AI Model** | Google Gemini API                        |
| **Email**    | Nodemailer (Gmail) or Resend             |
